# Gemini 整理摘要

来源：<https://gemini.google.com/share/658c6fd900e8>

该分享页围绕一个 PyQt6 四象限任务管理器反复迭代，核心结论已经整理进当前仓库。

## 已合并的设计点

- 自动排序：任务按完成状态、是否有截止时间、截止时间排序。
- 任务卡片：支持标题自动换行、描述悬浮提示、截止时间、提醒标记和完成复选框。
- 完成与归档：未完成任务显示在收件箱或四象限；今天完成的任务保留在原视图；更早完成的任务进入归档窗口。
- 归档窗口：打开后立即根据真实窗口宽度调整卡片，窗口尺寸变化时同步重新计算。
- 右键菜单：菜单样式集中到配置中，悬停时使用浅色背景和深色文字。
- 样式集中：卡片、按钮、菜单、列表、输入框等样式统一托管在 `app/config.py`。
- 多语言：所有界面文字放入 `app/resources/strings.py`，语言由 `app/config.py` 的 `LANGUAGE` 控制。
- 资源：应用 Logo、功能图标、复选框 SVG 放在 `app/resources/imgs/`。

## 整理策略

Gemini 对话中包含多个中间版本和局部替换片段。本仓库没有逐字复制每个片段，而是按最后状态重建为一套一致的工程结构，避免保留已经被后续修正覆盖的旧代码。
